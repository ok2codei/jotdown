# JotDown

A modern full-stack note taking application built for capturing ideas, organizing thoughts, and staying productive.

Live Demo: https://jotdown-eta.vercel.app/

---

## Features

- Authentication system
- Create, edit, and delete notes
- Organized note management
- Dark / Light theme support
- Protected routes
- Fast and responsive UI
- Mobile-friendly design
- Modular architecture
- Clean modern interface

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

### Clone the repository

git clone https://github.com/your-username/jotdown.git


### Navigate to project folder
cd jotdown
### Install dependencies
npm install
### Start development server
npm run dev
## Environment Variables

Create a `.env` file in the root directory.

```env
VITE_API_URL=your_api_url
```
